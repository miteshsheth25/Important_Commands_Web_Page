Basics

1. echo :- is used to display text or data on a web page or in the command line. It outputs strings or variables directly to the browser and is commonly used to
             show dynamic content like form results or HTML. It's a fundamental tool in PHP for creating dynamic web applications.

<?php
   echo "<h4>Welcome to My Website</h4>";
?>

2. print :- is used to output text or data to the screen, similar to echo. However, it has some key differences:

1) print accepts only one argument, while echo can take multiple.
2) print returns a value (1) after execution, making it usable in expressions, whereas echo does not return a value.

For eg:- print "text to be displayed";

3. if :- statement is a conitional statement that is used to execute code only if a certain condition is met.

if (condition) {
    // code to be executed if condition is true
}

4. else :- statement is often used in conjunction with the “if” statement to execute code when the condition is false. The basic syntax of the “if-else” statement is as follows:

if (condition) {
    // code to be executed if condition is true
} else {
    // code to be executed if condition is false
}

5. switch - statement is anoter type of conditional statement that can be used to execute different blocks of code based on te value of the variable or expression.

switch (expression) {
    case value1:
        // code to be executed if expression is equal to value1
        break;
    case value2:
        // code to be executed if expression is equal to value2
        break;
        // more cases can be added as needed
    default:
        // code to be executed if expression does not match any of the cases above
        break;
}

6. for - The “for” loop is a type of iteration statement that allows you to execute a block of code a specific number of times.

for (initialization; condition; increment) {
    // code to be executed
}

  • Initialization: Sets the starting value of a counter variable.
  • Condition: A boolean expression checked before each iteration; if false, the loop stops.
  • Increment: Updates the counter after each loop cycle.

7. while - This loop runs a block of code as long as a given condition is true. The condition is evaluated before each iteration, and if it's false, the loop stops.

while (condition) {
    // code to be executed
}

8. foreach - loop is used to iterate over arrays or other iterable structures. It automatically goes through each element, assigning its value to a variable during each iteration.

<?php
$fruits = ["apple", "banana", "cherry"];

foreach ($fruits as $fruit) {
    echo $fruit .;
}
?>

9. function - is a reusable block of code that performs a specific task. You define it once and can call it multiple times throughout your program. Functions help
                 make your code modular, organized, and easier to maintain by encapsulating logic into named sections..

<?php
   function greet($name) {
      echo "Hello, $name!";
   }

   greet("Alice");
?>

10. include/require - are commands that allow you to include external files in your PHP script.

<?php
   require 'header.php';   // Script stops if header.php is missing
   include 'menu.php';   // Script continues even if menu.php is missing
?>

11. Embedding PHP in HTML - PHP code can be embedded within HTML using the standard PHP tags The below statement dynamically inserts a heading into the HTML document.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>PHP Syntax Example</title>
</head>
<body>
    <h1><?php echo "Hello, PHP!"; ?></h1>

</html>

                                            the php.ini file to 'on'.

<?
// Here echo command will only work if setting is done as said before

13. Single Line Comment - This is single line or short relevant explanations that one can add to their code. To add this, we need to begin the line with (//) or
                                    (#).

//   This is a single line comment
#   This is also a single line comment

14. Multi Line COmment - is used to accomodate multiple lines with single tag and can be extended to many lines as required.To add this, need to begin and                                     end the line with (/*...*/).

/* This is a multi line comment
   In PHP variables are written
   by adding a $ sign at the
   beginning. */

15. Data Types :- PHP supports several data types, including

  • String :- A sequence of characters.
  • Integer :- Whole numbers.
  • Float (Double) :- Numbers with a decimal point.
  • Boolean :- Represents true or false.
  • Array :- A collection of values.
  • Object :- An instance of a class.
  • NULL :- A special type representing a variable with no value.
  • Resource :- A special type that holds a reference to external resources (like database connections).

Arrays

16. Indexed Arrays :- use numeric indexes starting from 0. These arrays are ideal when you need to store a list of items where the order matters.

<?php
   $fruits = array("apple", "banana", "cherry");
echo $fruits[0]; // Outputs: apple
?>

17. Associative Arrays :- use named keys, which are useful when you want to store data with meaningful identifiers instead of numeric indexes.

<?php
    $person = array("name" => "GFG", "age" => 30, "city" => "New York");
echo $person["name"];
?>

18. Multidimensional Array :- are array that contain other arrays as elements. These are used to represent more complex data structures, such as matrices or tables.

<?php
    $students = array(
    "Anjali" => array("age" => 25, "grade" => "A"),
    "GFG" => array("age" => 22, "grade" => "B")
);
echo $students["GFG"]["age"];
?>

19. array_push() :- Adds elements to the end of an indexed array.

$fruits = ["apple", "banana"];
array_push($fruits, "cherry");   // Adds "cherry" to the end

20. array_unshift() :- Adds elements to the beginning of an indexed array.

array_unshift($fruits, "pear");   // Adds "pear" to the beginning

21. Direct assignment :- Adds an element to an associated array.

$person["city"] = "New York";

22. array_pop() :- Removes the last element from an indexed array.

array_pop($fruits);

23. array_shift() :- Removes the first element from an indexed array.

array_shift($fruits);

24. unset() :- Removes a specific element from an array by key or index.

unset($fruits[2]);   // Removes the element with index 2

Array Functions

25. array_merge() :- This function combines 2 or more arrays into one.

$array1 = [1, 2, 3];
$array2 = [4, 5, 6];
$merged = array_merge($array1, $array2);
print_r($merged);   // Outputs: [1, 2, 3, 4, 5, 6]

26. in_array() :- This function checks if a specific value exists in an array.

$fruits = ["Apple", "Banana", "Cherry"];
if (in_array("Banana", $fruits)) {
   echo "Banana is in the array!";
}

27. sort() :- This function sorts an indexed array in ascending order.

$numbers = [3, 1, 4, 1, 5];
sort($numbers);
print_r($numbers);   // Outputs: [1, 1, 3, 4, 5]

Other Codes

28. Class :- A class defines the structure of an object, it contains variables and functions.

<?php
class GeeksforGeeks
{
   // Constructor
   public function __construct(){
     echo 'The class "' . __CLASS__ . '" was initiated!';
   }
}
// Create a new object
$obj = new GeeksforGeeks;
?>

29. Constructor :- A constructor is a special function or method in a class that is automatically called when an object of the class is created. The constructor is                          mainly used for initializing the object.

<?php
class Car {
   public $model;
   public $color;
   // Constructor
   public function __construct($model, $color) {
      $this->model = $model;
      $this->color = $color;
   }
   public function display() {
      echo "Model: $this->model, Color: $this->color";
   }
}
$myCar = new Car("Tesla", "Red");
$myCar->display();
?>

29. Destructor :- A destructor is a special method that is automatically called when an object is destroyed or goes out of scope. It is mainly used for cleaning up                        or releasing resources, such as closing file handles or database connections.

<?php
class Database {
   public $connection;

   // Constructor to initialize the connection
   public function __construct($hostname) {
      $this->connection = "Connected to database at $hostname";
      echo $this->connection;
   }

   // Destructor to close the connection
   public function __destruct() {
      echo "\nConnection closed.";
   }
}

$db = new Database("localhost");
?>

30. Access Modifier :- They control how and where the properties / mentods of a class can be accessed.There are 3 types :-
     public, protected and private

1. public :- can be accessed from anywhere, from within the class, by inherited (child) classes, and from outside the class.. If a property is declared    public, its value can be read or changed from anywhere in your script. This is Default access specifier

   <?php
   class Demo {
     public $name = "GeeksforGeeks";

     public function showName() {
       return $this->name;
     }
   }

   $obj = new Demo();
   echo $obj->name . "\n";
   echo $obj->showName();
   ?>

2. private :- can be accessible only within the class that declares it. It is not accessible in child classes or from outside the class.

   <?php
   class Myclass {
     private $secret = "Top Secret";

     private function getSecret() {
       return $this->secret;
     }

     public function revealSecret() {
     return $this->getSecret();   // Allowed internally
     }
   }

   $obj = new MyClass();
   echo $obj->revealSecret();   // Works
   echo $obj->secret;   // Error: Cannot access private property
   echo $obj->getSecret();   // Error: Cannot access private method
   ?>

3. protected :- can only be accessed within the class itself and by inheriting classes (subclasses). It is not accessible from outside the class.

   <?php
   class ParentClass {
     protected $message = "Hello from Parent";

     protected function showMessage() {
       return $this->message;
     }
   }
   class ChildClass extends ParentClass {
     public function getMessage() {
       return $this->showMessage();
     }
   }

   $obj = new ChildClass();
   echo $obj->getMessage();
   echo $obj->message;
   ?>

3 ways of working with MySQl and PHP

31. Using MySQLi object-oriented procedure :- can use the MySQLi object-oriented procedure to establish a connection to MySQL database from a PHP script.

<?php
    $servername = "localhost";
    $username = "username";
    $password = "password";

    // Creating connection
    $conn = new mysqli($servername, $username, $password);

    // Checking connection
    if ($conn->connect_error) {
       die("Connection failed: " . $conn->connect_error);
    }
    echo "Connected successfully";
?>

32. Using MySQLi procedural procedure :- is a procedural approach of MySQLi to establish a connection to MySQL database from a PHP script as described below.

<?php
    $servername = "localhost";
    $username = "username";
    $password = "password";

    // Creating connection
    $conn = mysqli_connect($servername, $username, $password);

    // Checking connection
    if (!$conn) {
       die("Connection failed: " . mysqli_connect_error());
    }
    echo "Connected successfully";
?>

33. Using PDO procedure (PHP Data Objects) :- in this method we connect to the database using data objects in PHP.

<?php
    $servername = "localhost";
    $username = "username";
    $password = "password";

    // Creating connection
    $conn = mysqli_connect($servername, $username, $password);

    try {
       $conn = new PDO("mysql:host=$servername;dbname=myDB", $username, $password);
       // setting the PDO error mode to exception
       $conn->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
       echo "Connected successfully";
       }
    catch(PDOException $e)
       {
       echo "Connection failed: " . $e->getMessage();
       }
?>

34. 3 ways to Closing A Connection.

    1) Using MySQLi object oriented procedure :- $conn->close();

    2) Using MySQLi procedural procedure :- mysqli_close($conn);

    3) Using PDO procedure :- $conn = null;
