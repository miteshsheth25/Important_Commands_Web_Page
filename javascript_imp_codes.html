Console & Debugging

1. To print messages or variables to the console.

console.log()

2. To display an error message in the console.

console.error()

3. To show warning message.

console.warn()

4. To display array / object data as table.

console.table()

5. To Pause JavaScript execution for debugging in developer tools.

debugger;
